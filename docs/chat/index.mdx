## Shared validation rules

1. `model` must be non-empty.
2. `messages` must contain at least one message.
3. Every message must have non-empty `role` and `content`.

## Streaming type

`StreamCompletions` returns `*ChatStream` with:

- `Next()`
- `Current()`
- `Choice()`
- `Text()`
- `Err()`
- `Close()`
