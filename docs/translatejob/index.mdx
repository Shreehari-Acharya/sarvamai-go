## Workflow

1. Initialize job.
2. Fetch upload links.
3. Upload files.
4. Start job.
5. Poll status.
6. Fetch download links.

## Important SDK validation note

Validation in this package is intentionally minimal. SDK checks only required IDs/files and callback URL presence, but does not enforce model or diarization combinations client-side.
